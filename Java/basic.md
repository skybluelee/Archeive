# javac
자바로 코딩한 .java 파일을 자바 바이트코드로 compile한다.
```
$ ls                                                     
Main.java

$ javac Main.java

$ ls                                                     
Main.class  Main.java
```
`class` 파일이 추가로 생성된다.

`class` 파일은 바이트 코드를 사용자가 해석할 수 있도록 디코딩한 파일이다.
# java
```
$ java Main
Hello world
```
`.java` 파일을 실행한다.
